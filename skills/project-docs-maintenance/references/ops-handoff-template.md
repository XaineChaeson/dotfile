# Ops Handoff Checklist

Use this for on-call handoff and operational readiness.

## Current status
- Service health summary.
- Known incidents or risks.

## Upcoming changes
- Deployments scheduled.
- Feature flags or migrations in flight.

## Runbooks and owners
- Primary runbook links.
- On-call rotation or owner contacts.

## Key metrics
- Dashboards and alert thresholds.
